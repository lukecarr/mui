# Security Policy

## Supported Versions

In `v0.x.x` versions (pre-stable), we do not commit to supporting any historic
(non-latest) versions with security updates.

## Reporting a Vulnerability

We have "private vulnerability reporting" enabled for the MUI repository.

You can privately (and responsibly) disclose a security vulnerability to MUI's
maintainers by following [these GitHub docs instructions][docs].

[docs]: https://docs.github.com/en/enterprise-cloud@latest/code-security/how-tos/report-and-fix-vulnerabilities/privately-reporting-a-security-vulnerability
